# PlacementGPT - Your AI Placement Mentor

![PlacementGPT](https://images.unsplash.com/photo-1531746020798-e6953c6e8e04?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&q=80)

PlacementGPT is a futuristic, AI-powered placement preparation platform designed to help students master their interviews. Built with modern web technologies, it features dynamic glassmorphism UI, neon cyber-aesthetics, and highly interactive simulated AI components.

## 🚀 Features

* **Authentication & Dashboard:** Secure local login flow with a dynamic dashboard, notification history, and profile management.
* **Resume Analyzer:** Upload your resume (PDF) to receive an instant simulated AI breakdown of your technical skills, missing frameworks, and ATS formatting errors.
* **Mock Interviews:** Conversational AI mentor that uses Speech-to-Text to listen to your answers, evaluates them based on your chosen domain (Web, Python, Java), and responds with context-aware feedback.
* **Dynamic Coding Roadmaps:** Generates a highly specific, 3-phase curriculum based on your target role, dream company, and current level.
* **Aptitude Tests:** 30-minute rigorous mock assessments that auto-submit and track performance.
* **HR Answer Enhancer:** Captures your exact words and dynamically rewrites them into impactful, grammar-perfect professional answers.

## 🛠️ Technology Stack

* **Frontend:** HTML5, Vanilla JavaScript, Modern CSS3
* **Design System:** Custom CSS tokens, Glassmorphism, Neon Gradients, Animations
* **APIs Used:** Web Speech API (Speech Recognition & Synthesis)

## 💻 How to Run Locally

Since this is a lightweight, frontend-first prototype, no complex build steps are required!

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/PlacementGPT.git
   ```
2. Navigate into the project directory:
   ```bash
   cd PlacementGPT/frontend
   ```
3. Open `index.html` directly in your browser, or serve it via a local static server:
   ```bash
   npx serve .
   ```
